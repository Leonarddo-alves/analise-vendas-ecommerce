# Análise de Vendas E-commerce

Este projeto apresenta uma análise exploratória de dados de vendas, com o objetivo de identificar padrões de faturamento por categoria, região, produto, mês e segmento de cliente.

## Objetivo

Analisar dados de vendas para entender quais fatores mais contribuem para o faturamento total.

## Ferramentas utilizadas

- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab
- GitHub

## Perguntas respondidas

- Qual foi o faturamento total?
- Qual categoria teve maior faturamento?
- Qual região vendeu mais?
- Quais produtos geraram mais receita?
- Como as vendas variaram ao longo dos meses?
- Qual segmento de cliente teve maior participação nas vendas?

## Principais resultados

- O faturamento total foi de aproximadamente USD 2,26 milhões.
- A categoria com maior faturamento foi Technology, com cerca de USD 827 mil.
- A região com maior faturamento foi West, com aproximadamente USD 710 mil.
- O produto com maior faturamento foi Canon imageCLASS 2200 Advanced Copier, com cerca de USD 61,6 mil.
- O segmento Consumer foi o principal segmento, com aproximadamente USD 1,15 milhão em vendas.
- As vendas apresentaram maior força em alguns meses do segundo semestre, especialmente entre setembro e dezembro.

## Conclusão

A análise indica que produtos de tecnologia, clientes do segmento Consumer e a região West tiveram maior impacto no faturamento total.

Esses resultados podem ajudar a empresa a entender melhor seus principais motores de receita e investigar oportunidades de melhoria em regiões ou segmentos com menor desempenho.
