# Análise de Vendas E-commerce

Este projeto tem como objetivo analisar dados de vendas de um e-commerce para identificar padrões, produtos mais vendidos e oportunidades de melhoria.

## Objetivo

Praticar análise exploratória de dados usando Python, Pandas e visualização de dados.

## Ferramentas utilizadas

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Perguntas da análise

- Qual foi o faturamento total?
- Quais produtos venderam mais?
- Qual categoria teve maior faturamento?
- Qual mês teve mais vendas?
- Existe algum padrão nos dados?
